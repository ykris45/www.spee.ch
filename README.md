## Prerequisites

1. You will need the following tools installed:

* [Node.js](https://nodejs.org/en/download/)
* NPM (should come installed with node)
* [Git](https://git-scm.com/downloads)
* Curl
* Tmux
* Unzip

2. Make sure npm is up-to-date
```
$ npm update
```
